# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in this package, please report it responsibly:

1. **Do NOT open a public issue.**
2. Email **security@flowdelegate.com** with details of the vulnerability.
3. Include steps to reproduce if possible.

We will acknowledge your report within 48 hours and aim to release a fix within 7 days for confirmed vulnerabilities.

## Scope

This is a pure-data utility library with zero runtime dependencies and no network access. The attack surface is minimal, but we still take all reports seriously.

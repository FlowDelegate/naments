# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-08

### Added
- Initial release
- 10 themed categories: 80s, 90s, cats, christmas, easter, movie-stars, presidents, simpsons, south-park, super-heroes
- Core API: `getNames`, `getName`, `getCategories`, `hasCategory`
- Dual ESM/CJS output with TypeScript declarations
- `npm run new-category` scaffolding script
- CI pipeline testing Node 18, 20, and 22
- CODEOWNERS requiring maintainer review on all PRs
